Privacy & Security

Your files stay yours

No file ever has to leave your computer.

Browser-only processing
All PDF operations run in JavaScript inside your browser tab. No backend servers involved.
Zero uploads
Files are read locally via the File API. Nothing is transmitted over the network.
No storage
Files exist only in memory while you process them. Closing the tab removes all data.
Open libraries
Powered by pdf-lib and PDF.js — open-source, auditable, trusted by millions.
